# Nuxt.js
## 개요
* Vue.js 애플리케이션 **서버 렌더링**을 하기 위한 구성이 미리 설정 되어 있음
* nuxt generate 이라고 부르는 Vue.js 애플리케이션을 정적으로 생성 하는 배포 옵션을 제공함
* 포함된 기능
  * Vue 2
  * Vue Router
  * Vuex
  * Vue Server Renderer
  * vue-meta

## Setup
```sh
> npx create-nuxt-app <project-name>
```
또는
```sh
> yarn create nuxt-app <project-name>
```

## Development
```sh
> yarn dev
```

## Ref
https://nuxtjs.org/
